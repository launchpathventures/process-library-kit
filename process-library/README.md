# Process Library

Repeatable processes extracted from real work. Each includes a definition, gold standard example, and evaluation rubric.

## Commands

| Command | What it does |
|---------|-------------|
| `/extract-process` | Analyze completed work and capture as a reusable process |
| `/run-process {name}` | Execute a process with gold-standard evaluation |
| `/list-processes` | Browse all available processes |

## How Processes Improve

1. **Run records** track scores over time — declining scores signal the process needs updating
2. **Gold standards can be replaced** when a better example emerges
3. **Steps refine** when runs reveal missing or unnecessary steps
4. **Evaluation criteria sharpen** when scoring reveals ambiguous dimensions
